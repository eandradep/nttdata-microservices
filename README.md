## Cómo empezar

### Instalación

Para empezar, necesitas instalar todos los requisitos mencionados en la sección anterior en tu sistema. Cuando estén correctamente instalados, puedes clonar este repositorio a una carpeta local:

mvn clean install

docker-compose up
